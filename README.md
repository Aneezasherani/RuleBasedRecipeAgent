# Recipe Recommendation Rule-Based Agent

## Description
This is a beginner-friendly rule-based AI agent that recommends recipes based on the ingredients you have. Users input available ingredients, and the agent suggests matching dishes with step-by-step instructions.

## Files
- `recipe_agent.ipynb` : Jupyter Notebook containing the Python code for the agent.
- `recipes.csv` : Dataset containing recipes, ingredients, steps, cuisine, meal type, and nutrition info.

## How to Use
1. Open `recipe_agent.ipynb` in Jupyter Notebook or VS Code with Jupyter extension.
2. Run the cells step by step.
3. Enter available ingredients as comma-separated values when prompted.
4. The agent will display dishes you can make along with cooking steps.

## Dataset
The dataset includes:
- Recipe Name
- Ingredients
- Steps
- Cuisine
- Meal Type
- Prep Time
- Cook Time
- Total Time
- Calories

## Notes
- Ensure that the CSV file is in the same folder as the notebook.
- Ingredients should be entered in lowercase and comma-separated for accurate matching.
